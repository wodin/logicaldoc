Parameters

Another system configuration page is Administration -> System -> Parameters. The user can specify:

  • 1) OCR resolution threshold
  • 2) OCR text threshold
  • 3) minimum password length. This parameter can be modified in the page Administration -> Security -> Security (tab Password)
  • 4) password time to live. This parameter can be modified in the page Administration -> Security -> Security (tab Password)
  • 5) server port number on which is listening the Open Office service for the converting document into Pdf format feature
  • 6) the minimum and maximum tag size
  • 7) download ticket time to live
