The LogicalDOC development team is proud to announce the immediate availability of LogicalDOC CE 6.6

You can get this distribution in tomcat-bundle archive from SourceForge: 
http://sourceforge.net/projects/logicaldoc/files/distribution/LogicalDOC%20CE%206.6/logicaldoc-6.6-tomcat-bundle.zip/download

LogicalDOC is a Web-based Document Management System (DMS) based on open source technology (GWT +Spring +Hibernate +Lucene). 
Thanks to its intuitive interface, LogicalDOC transforms complex operations into easy tasks. 
It allows the storage, organization, versioning and use of all kind of files and documents within an user friendly interface.

New Features
Customizable User Dashboard

Improvements
Improved performance of massive folders deletions
Support for the Swedish language
Improved performance of the preview on multi-page documents
Replaced the Flash used to display the Repository Statistics with a JavaScript component
Replaced the Flash used to display the Tag Cloud with a JavaScript component
