LDAP

LogicalDOC è in grado di gestire l'autenticazione degli utenti in varie modalità in aggiunta a quella standard precedentemente descritta (paragrafo Utenti). Infatti LogicalDOC può interagire sia con un sistema di Active Directory di MS Windows, sia con un sistema di autenticazione standard come LDAP. Accedendo alla pagina Amministrazione --> Sicurezza --> LDAP è possibile configurare i parametri di connessione ad un sistema di autenticazione esterno, fra i quali:

  • 1) URL (comprensivo del numero di porta per poter accedere al sistema di autenticazione)
  • 2) Username
  • 3) Password
  • 4) Nome del dominio
  • 5) DN, dove poter inserire la OU dove si trovano gli utenti, ad esempio OU=Utenti,DC=dominio,DC=dom
