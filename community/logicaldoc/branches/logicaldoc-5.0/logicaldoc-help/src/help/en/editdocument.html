Modifying documents

View the menu Tools of document positioning itself on the name of the document (the document is highlighted in blue) and then click the right mouse button.


The Edit page is accessible by selecting "Edit" from the document's contextual menu.

The editing mask is basically the same as the one used to make insertions but this time only metadata fields are editable
In particular, the editable properties are the document name, key words, source, author, coverage, document type, date and language.
It is possible to add key words in the available field by separating them by commas.

Note: it is also possible to establish a logical relation with other documents by specifying on each one of them the same key word, e.g.: if we would like to connect documents of different folders but from the same project, they will become related each other after assigning the key word "project754" to everyone. Using the search by key word function, it is possible to show inside one view all the related documents.
