Basic knowledge

Login

In order to accces the system, it is necessary an user name and a password. I you do not have any of them, ask your system Administrator for one.
In the Login page, one may choose the working language among the following ones: English, German, Italian, Spanish, French.
If you select the "Default" option, the system will use the language configured as user preference.

Home

After logging in, you can see the personal homepage, from which you can see Tag Clouds (keywords used), the latest documents Modified or Downloaded.
Also from the homepage you can access the messages received and create a new message.

Main menu


The upper image shows the words from the main menu. Each user can see only those menus in which he has access rights.
Once positioned on the menu using the cursor over the desired word, this will become highlighted automatically.
Clicking on a menu word the corresponding page is shown.

System messages

This application consists in reception and forwarding of messages to other system users.
This functionality is available either using the link present on the home page (Create messages - Read messages) or from the menu: Personal Data --> Messages.

Simple search

It is possible to make a search over the documents archive using the form present on the page upper side.
If you remember any word present in the recently uploaded documents text, write it in the searching edit box.
Clicking the search icon (magnifying glass), search by given word is made.
