
LogicalDOC is a Web-based Document Management System (DMS), easy to use and learn. 

This 4.5 release is an improved version of LogicalDOC.

The file system structure was completely revised.

It was also introduced support for LDAP (thanks to Sebastian Wenzky).

Among other things various optimizations has been implemented.

LogicalDOC uses the following core technologies:

 * J2EE 2.4 web tier
 * JSF 1.2
 * JAX-WS
 * Lucene 2.4
 * Hibernate
 * Spring
 * CXF
 * ICEfaces
 * JPF

For details of changes see the ChangeLog.
