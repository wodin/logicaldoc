Security on Menus

In LogicalDOC you can define security policies on each menu.
In fact, you can determine which entities (users and groups) can see a menu. By clicking the Browse... button, you can see a window that shows all the menus in LogicalDOC. After selecting a menu, press OK then you can see the entities which can see this menu.
You can move an entity from the list of those available to the list of those assigned using the >> button. If you want to bring back an entity to the ‘available’ list click the << button. If you want to select multiple entities, hold down the SHIFT key and select your items.
