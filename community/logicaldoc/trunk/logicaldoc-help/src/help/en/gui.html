GUI

The user can configure the LogicalDOC graphical interface accessing the page LogicalDOC Administration -> System -> GUI. The user, in fact, can choose:

  • 1) the home page displayed after the user login
  • 2) the default visualization for documents list page
  • 3) the default visualization for documents list after a documents search
  • 4) the icon size when the user chooses the icons visualization for the documents list page
  • 5) the number of documents in a single document list page
