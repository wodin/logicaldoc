WebDAV

Through the WebDAV feature the user can see, manage, move, copy, save and rename files and folders in LogicalDOC as if they were directly on your personal computer. The page Administration -> Client Tools -> WebDAV allows the user to manage the LogicalDOC WebDav feature. The user, through the Status checkbox, can enable or disable the funcionality, and he can visualize the URL on which the Web Service have been installed.
