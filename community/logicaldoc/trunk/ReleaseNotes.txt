The LogicalDoc team is pleased to announce the availability of the new release 
of LogicalDOC Enterprise, that brings you new and exciting features.

Among other things, these are the major new features:

Workflow: Visual Business Process editor directly accessible from the web browser
Workflow: Multiple task assignments
Workflow: Support for activities start and stop
History on document folders
New graphical layout
Document navigation through a folders tree
Configurable first version number
3D (Cumulus) Tag Cloud
New graphical layout more lighter and sleek
Improvements in the ergonomics of the GUI

For details of changes see the ChangeLog.
