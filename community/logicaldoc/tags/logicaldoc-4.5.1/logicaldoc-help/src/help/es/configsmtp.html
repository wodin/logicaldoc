Configurar preferencias smtp

El sistema ofrece la posibilidad de enviar documentos mediante e-mail.
Tales funciones, permiten esencialmente generar una correo, para una dirección específica, consistente en alguna información referente al documento seleccionado.

Para poder realizar tales funciones, es indispensable configurar las preferencias del servidor smtp (servidor de correo de salida) del sistema.

Accede a la ventana de configuración mediante la ruta: Administración --> E-Mails --> SMTP.

Todos los parámetros del formulario son obligatorios.
Nota: La contraseña se muestra en claro para faciliar la administración.

Para verificar la correcta configuración, se aconseja verificar el envío de un documento mediante e-mail.
