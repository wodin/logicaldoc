Customize smtp

The system offers the possibility to send documents via email.
Such functionality essentially allows to generate a mail to specific addresses containing in it the selected document.

For being able to take advantage of such functionality it is indispensable to set up an smtp server (mail server) on the system.

Access therefore the configuration mask using the main menu path: Administration --> System --> SMTP .

All the fields of the form are mandatory.

Nota:  
To verify if the configuration is correct, you should check the sending of a document via e-mail.
