Importare Documenti da archivio .ZIP

In alcuni casi può essere utile poter inserire sul sistema in un'unico passaggio un numero cospicuo di documenti mantenendo una specifica organizzazione in cartelle.
In LogicalDOC questa funzionalità è disponibile nella forma di importazione da file archivio .ZIP.
Grazie a questa funzione è possibile replicare sul sistema l'esatta struttura della cartella "zippata", sia per i nomi e gli annidamenti delle cartelle sia con i nomi ed il posizionamento dei documenti in esse contenuti.

Occorre tener presente che esistono comunque delle limitazioni, la pricipale delle quali è che il file .ZIP da caricare non deve essere eccesivamente grande.
Si consiglia di non eccedere la dimensione di 20 MB (megabyte).

Per prima cosa l' utente deve creare la cartella con i documenti sul proprio PC.
Successivamente l' utente deve produrre un archivio .ZIP della cartella, utilizzando uno strumento come WinZip, 7-Zip o similari.
Infine il file ZIP creato deve essere caricato in LogicalDOC.

Caricamento del file .ZIP

  1. Nell'area Documenti selezionare la cartella "padre" che ospiterà i documenti contenuti nell'archivio.
    Il percorso corrente viene mostrato nella parte destra dello schermo.



  2. Cliccando sull'icona "Importa archivio zip" apparirà una maschera per l'upload di file simile a quella per l'inserimento di un nuovo documento.
  3. Premere il tasto "Sfoglia" e selezionare sul disco fisso locale un file archivio .ZIP.
  4. Premere il tasto "Upload" per trasferire il file su LogicalDOC.
    La barra indicatrice mostra l'avanzamento della operazione di Upload (trasferimento file).
    Al termine, quando la barra avrà raggiunto il 100%, apparirà il pulsante "Importa archivio zip" a destra nella maschera di inserimento.
  5. Premere il pulsante "Importa archivio zip" per avviare l'importazione.



L'indicatore dello stato (in alto a destra) prende a lampeggiare segnalando che una operazione è stata avviata.
Dopo un certo periodo (che potrebbe essere anche di alcuni minuti) viene mostrata la cartella "padre" con i nuovi contenuti.

Una volta avviata l'operazione l'utente deve attendere il completamento dell'importazione. Poiché questa potrebbe essere di lunga durata la sessione utente potrebbe interrompersi, in questo caso non c'è da preoccuparsi, il sistema comunque procederà con l'importazione, sarà solamente necessario rieseguire la login in un secondo momento.

LogicalDOC adotterà la struttura della cartella "zippata" e copierà tutti i documenti nell'archivio.
Ciascuna nuova cartella e ogni documento erediteranno tutti i permessi della cartella entro la quale verranno importati.
